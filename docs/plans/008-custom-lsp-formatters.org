* Motivation

LSP servers and formatters are hardcoded in ~builtin_configs()~ and
~register_builtins()~. Users working with languages not covered by the builtins
(C/C++, Elixir, Zig, etc.) can't get diagnostics or formatting. The config file
from plan 007 already defines the schema — this plan wires it into the LSP and
formatter systems.

* Scope

Make ~LspServerConfig~ and ~FormatterConfig~ accept ~String~ (not ~&'static str~)
so they can be constructed from config file data. Add methods to register custom
entries from the parsed config.

* Current State

- ~LspServerConfig~ (kodo-lsp/config.rs:1-12) uses ~&'static str~ for all
  string fields. Cannot be constructed from runtime data.
- ~FormatterConfig~ (kodo-fmt/registry.rs:5-13) already uses owned ~String~.
  Already supports ~register()~. Ready for custom entries.
- ~builtin_configs()~ returns ~Vec<LspServerConfig>~ with 4 hardcoded entries.
- ~register_builtins()~ in ~FormatterRegistry~ adds 4 formatters.

/// Register an additional LSP server configuration.
/// Call after construction to add servers from config files.
pub fn register_server(&mut self, config: LspServerConfig) {
    if is_command_available(&config.command) {
        tracing::debug!(name = %config.name, "registering custom LSP server");
        // Store the config for this server
        // (implementation depends on how LspManager stores configs internally)
        for ext in &config.extensions {
            self.extension_map.insert(ext.clone(), config.clone());
        }
    } else {
        tracing::debug!(name = %config.name, command = %config.command, "custom LSP server not available on PATH");
    }
}
#+end_src

/// Register custom formatters from config.
fn register_custom_formatters(
    registry: &mut FormatterRegistry,
    config: &FormattersConfig,
) {
    for (name, entry) in &config.formatters {
        registry.register(FormatterConfig {
            name: name.clone(),
            command: entry.command.clone(),
            extensions: entry.extensions.clone(),
        });
        tracing::debug!(name, "registered custom formatter");
    }
}

/// Register custom LSP servers from config.
fn register_custom_lsp_servers(
    lsp_manager: &mut LspManager,
    config: &LspConfig,
) {
    for (name, entry) in &config.servers {
        let server_config = LspServerConfig {
            name: name.clone(),
            command: entry.command.clone(),
            args: entry.args.clone(),
            extensions: entry.extensions.clone(),
        };
        lsp_manager.register_server(server_config);
    }
}
